⚠ Notice This page is a launcher guide — it cannot start the backend or frontend by itself. You must run the launch script first, then click Open Dashboard below.
Prerequisites Please ensure the following are installed before running any launch script:
  • Node.js 18+ — https://nodejs.org/
  • Python 3.10+ — https://www.python.org/downloads/
  • First run will auto-install npm & Python dependencies (may take a few minutes).

02 — Start the App
macOS
Double-click start-mac.command
If macOS blocks execution, run once in Terminal:
chmod +x start-mac.command
Windows
Double-click start-windows.bat
Two separate windows will open — one for the backend, one for the frontend. Keep both running.
03 — First-Time Setup
  1. Download or clone the repository from GitHub.
  2. Run the launch script for your platform (macOS / Windows above).
  3. Wait for dependencies to install automatically on first run.
  4. Open http://localhost:5173 in your browser.
  5. Click the ⚙ Settings icon and enter your DeepSeek API Key.
    Never paste your API key into this HTML file or into the launch scripts.
  6. Set Initial Total Credit to estimate your historical spend.
04 — Security
  • 🔒   Your API Key is stored only in backend/local_settings.json on your machine.
  • 🔒   backend/.env and backend/local_settings.json are in .gitignore — they will not be committed.
  • 🔒   Never share your API Key. Never upload .env or local_settings.json to GitHub.
  • 🔒   This app is local-only. No data leaves your machine except the DeepSeek balance API call.
